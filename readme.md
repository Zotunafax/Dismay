# Dismay

Dismay is a wip multifunctional Android application.

## Naming convention

### Layout Files
Layout files are named like "activity_[function of screen].xml".
The second part should be used for ids in that specific file. The id also contains the type of the element and a one word description of the function of the element.

e.g. file name = "activity_main.xml" -> id = "main_textView_header"
